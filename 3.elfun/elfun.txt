help sin
sin - 参数的正弦，以弧度为单位

    此 MATLAB 函数 返回 X 的元素的正弦。sin 函数按元素处理数组。该函数同时接受实数和复数输入。 对于 X 的实数值，sin(X) 返回区间 [-1,
    1] 内的实数值。 对于 X 的复数值，sin(X) 返回复数值。

    Y = sin(X)

    另请参阅 <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/asin.html">asin</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/asind.html">asind</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/sind.html">sind</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/sinh.html">sinh</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/double.sinpi.html">sinpi</a>

help angle
angle - 相位角

    此 MATLAB 函数 为复数数组 z 的每个元素返回区间 [-π,π] 中的相位角。theta 中的角度表示为 z =
    abs(z).*exp(i*theta)。

    theta = angle(z)

    另请参阅 <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/abs.html">abs</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/atan2.html">atan2</a>, <a href="https://www.mathworks.com/help/releases/R2020b/matlab/ref/unwrap.html">unwrap</a>
